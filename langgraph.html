① StateGraph — Cấu trúc Graph
LangGraph định nghĩa pipeline dưới dạng StateGraph — một đồ thị có hướng (DAG) gồm các Nodes (hàm xử lý) và Edges (luồng chuyển tiếp). Toàn bộ dữ liệu được gom vào một object duy nhất là PipelineState (TypedDict) — mỗi node nhận state cũ, trả về dict chứa các field cần cập nhật, LangGraph tự merge. Không có biến global, không có side effect ngoài DB — mỗi bước hoàn toàn độc lập.
② interrupt_before — Human-in-the-loop
Graph được compile với tham số interrupt_before=["human_review"]. Khi pipeline chạy đến đây, graph.ainvoke() tự động return về mà không chạy human_review. Frontend polling phát hiện status="reviewing" → hiển thị UI review. Khi người dùng Phê duyệt, backend gọi graph.ainvoke(human_edits, thread_id=run_id) — graph resume đúng từ điểm bị dừng, không chạy lại các node trước.
③ MemorySaver — Checkpoint
MemorySaver là checkpointer mặc định — lưu toàn bộ PipelineState tại thời điểm interrupt vào RAM, key bởi thread_id = run_id. Khi resume, LangGraph load lại checkpoint, merge thêm human_edits vào state rồi tiếp tục chạy từ node_4. Lưu ý: MemorySaver lưu trong RAM — nếu container restart, các run đang "reviewing" sẽ mất state. Nâng cấp lên PostgresSaver để persistent.
④ Node 3 — write_fields · OpenAI loop
Node 3 là node nặng nhất — lặp tuần tự qua từng field tô vàng, gọi OpenAI GPT-4o với response_format={"type":"json_object"}. Mỗi lần gọi truyền vào: system prompt quy tắc văn phong + ngữ cảnh đoạn văn + placeholder + tối đa 14.000 ký tự tài liệu nguồn. GPT-4o trả về {"value", "confidence", "reason"} — confidence do AI tự đánh giá dựa trên mức độ tìm thấy thông tin trong tài liệu nguồn.
⑤ PostgreSQL — READ / WRITE tại từng node
Database được truy cập trực tiếp từ các nodes qua asyncpg — không đi qua LangGraph state để tránh state quá nặng. Node 1 READ template_fields → Node 2 READ source_documents → Node 3 WRITE field_results → Node 4 UPDATE field_results.final_value → Node 5 READ templates.file_data (BYTEA). File .docx gốc được lưu dưới dạng BYTEA trong DB, đảm bảo mỗi lần export đều dùng template nguyên bản.
